# java-cli-maven-surefire-jacoco-junit-test-car-data

## Description
A POC for sprimg app using JUnit
framework with jacoco and surefire
plugins.

## Tech stack
- java
- maven
  - spring
  - junit
  - surefire
  - jacoco

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- jacoco report under bin/target/site/jacoco

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
